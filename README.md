# Fasm.ga - 255

## Servizio
Fasm.ga 255 (https://255.fasm.ga) è il servizio di note dell'ecosistema Fasm.ga.
La sua particolarità consiste nel bloccare le note oltre 255 caratteri, i titoli oltre 30 e le descrizioni oltre 50.
